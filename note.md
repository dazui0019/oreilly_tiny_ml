## 线性回归

![线性回归](images\Okuns_law_quarterly_differences.png)
有些数据可能会满足线性分布, 比如ADC的采样值与温度值或者电压值理论上应该满足一个线性函数$y=k*x+b$, 但是由于采集误差等原因我们采集到的ADC值会离散的分布在这条线性函数的周围。并且由于传感器在不同电压/温度的情况下误差也会有所不同, 比如NTC在$10^\circ\mathrm{C}$和$100^\circ\mathrm{C}$下的误差肯定是不同的, 因此我们要找到一个误差最小的模型(**找到一条误差最小的直线**)。

我们把"实际数据"到"模型预测值"的均方差(因为两数之差会存在负数, 所以要平方一下), 作为我们模型的误差值。我们最终就是要确定模型 $y=a*x+b$ 中某个 $k$ 和 $b$ 的组合, 使得我们模型的误差最小。
$$Loss = \frac{1}{n} \sum_{i=1}^{n} (y_{预测}^{(i)} - y_{真实}^{(i)})^2$$
![误差](images\误差.png)

因为不同的 $k$ 和 $b$ 的组合(不同的直线), 都会得到一个误差值。但是这样的 $k$ 和 $b$ 的组合会有无数多个, 人类无法穷举出所有的情况。

| 直线编号 | 斜率($k$) | 截距($b$) | 误差($z$) |
| :---: | :---: | :---: | :---: |
| $1$ | $k_1$ | $b_1$ | $z_1$ |
| $2$ | $k_2$ | $b_2$ | $z_2$ |
| ... | ... | ... | ... |
| $n$ | $k_n$ | $b_n$ | $z_n$ |

所以误差值就是一个以直线斜率和截距为自变量的二元函数$z = \phi(k, b)$, 形状类似图片这样。我们需要找到$z$最小的那个点, 就是函数图形上最底部的点。

### 创建模型

使用 `TensorFlow` 的 `Keras` 工具。

``` python
# 创建一个空的序列模型框架
# 后面add()的神经元层会依次添加到该框架上
model_1 = tf.keras.Sequential()

# 往模型框架里面添加神经元层
# Dense表示全连接层, 前一级的输入会连接到Dense层的每一个神经元
# input_shape=(1,), 表示输入数据的维度, 这里是一个1维向量
model_1.add(keras.layers.Dense(8, activation='relu', input_shape=(1,)))
# 也是一个Dense层而且只有一个神经元, 前一级的8个神经元会全部连接到该神经元上
model_1.add(keras.layers.Dense(1))


model_1.compile(optimizer='adam', loss='mse', metrics=['mae'])
model_1.summary()
```

#### 激活函数

机器学习（尤其是监督学习）的本质确实可以看作是在高维空间中进行函数拟合。而对于现实生活中出现的情况, 肯定无法使用一个线性函数来拟合, 因此需要一个非线性的激活函数来让训练后的模型非线性化。

下面这张图是不加激活函数的情况, 每一个神经元的输出函数就只是 `input` 与 `weight` 和 `bias` 的线性关系 $y = wight*x+bias$。假如是一个两层神经元的结构, 那么将上一次神经元的输出函数作为下一个神经元的输入可以得到 $y = wight*(wight*x+bias)+bias$, 系数wight相乘还是一个常量, 所以这任然是一个线性函数。因此如果不引入激活函数, 那么即使增加再多层的神经元, 最终也只是一个线性函数。

![无激活函数](images\无激活函数.png)

但是激活函数也分好坏, 下图引入就是一个不好的激活函数 $\phi(x)=x^2$, 引入该函数会导致输入`x`传导至末端时变得非常大, 甚至让计算机无法计算, 这个叫 **数值爆炸**。
